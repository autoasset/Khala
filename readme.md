### 需要先安装 

```bash
brew install node
# SVG => PDF
brew install --cask inkscape
# GIF裁切
brew install vips
# png 转换压缩
brew install libpng
# 最后调
npm install 
```

卸载:

```
brew uninstall inkscape
brew cleanup
```

[inkscape](https://inkscape.org/doc/inkscape-man.html)
[imagemin](https://github.com/imagemin/imagemin)
[libpng](http://www.libpng.org/pub/png/)
